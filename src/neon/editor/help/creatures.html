Creatures

Add/remove creatures

Right click to add or remove creatures. When adding a creature, an id for the creature is requested. If a creature with the given id already exists, the creature will not be created. When a new creature is added, the creature editor window will open.

Edit creatures

Double click on a creature to open the creature editor window.

Name

By default, the name of the creature is the same as the id. If the name field is left empty, the creature id will also be used.
