Items

Add/remove items

Right click to add or remove items. When adding a item, an id for the item is requested. If a item with the given id already exists, the item will not be created. When a new item is added, the item editor window will open.

Edit items

Double click on a item to open the item editor window.

Name

By default, the name of the item is the same as the id. If the name field is left empty, the item id will also be used.
